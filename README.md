Dari analisis diatas didapatkan kesimpulan, bahwa program pemerintah daerah dalam menggencarkan vaksin [sebagai syarat administrasi publik dan berpergian](https://www.beritasatu.com/archive/807969/anies-tegaskan-vaksinasi-jadi-persyaratan-administrasi-berkegiatan-di-jakarta), terbukti mampu menurunkan nilai kematian per hari sebesar **86%** (terhitung dari hari terakhir penggambilan data). Karena vaksin meningkatkan respons kekebalan pada orang dewasa yang lebih muda dan lebih tua.
